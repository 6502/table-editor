table-editor

an HTML5 table editor

The editor is a single self-contained HTML file that allows creating and editing a table directly in the browser, and works also when opened from the local file system.

The editor is loosely based on how org-mode table editing works in emacs and lets you:

  • Adding/removing rows or columns
  • Moving columns and rows
  • Switching cells from TD to TH and vice-versa
  • Set cell content horizontal alignment
  • Merging cells to create group titles (horizontal only)

All changes are dynamically reflected in a text-area where HTML code is shown and can be used for clipboard copy
