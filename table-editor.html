Table editing commands:
  • Tab/Shift-Tab: move to next/previous cell
  • Ctrl-Left/Ctrl-Right: swap column with previous/next
  • Ctrl-Up/Ctrl-Down: swap row with previous/next
  • Ctrl-Enter/Ctrl-Shift-Enter: Add a row/column after current one
  • Ctrl-Backspace/Ctrl-Shift-Backspace: Removes current row/column
  • Ctrl-Space/Ctrl-Shift-Space: Increments/decerements colspan
